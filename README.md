# USI-LS汉化补丁
## 前言
USI Life Support的汉化版本，完成了所有面板和组件的汉化，如果存在英文，应该是USI旗下的其他mod未汉化导致的，已向原作者提交了合并申请，建议搭配Hanatl制作的[USI汉化包](https://tieba.baidu.com/p/10013226580?fid=3306110&pid=152934723265&cid=152936135938#152934723265)使用，同时感谢@GallyYoko2024对MKS本体的汉化，以及@tinygrox编写的[汉化教程](https://github.com/tinygrox/KSP_Mod_Localization_Guide)。如果在游玩的过程中遇到任何未汉化的部分或BUG欢迎随时向我反馈。

## 使用方法
下载本页面的zip文件，解压后将LifeSupport文件夹复制到游戏根目录下的GameData\UmbraSpaceIndustries下，并选择覆盖所有文件
